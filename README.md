## 🚀 Key Projects

### 🎨 Collaborative Pixel Art Editor
**C++ · Qt · QML · POSIX**

Multi-user real-time collaborative editor with a client-server architecture. Features layered rendering, thread-safe synchronization, and a tool dispatch system built with `std::variant`. Built as part of INFO-F209 (Systems Programming) @ ULB.

---

### 🗺️ RPG Map Editor
**C++ · Qt6 · QML**

Collaborative RPG map editor with real-time sync and role management. Supports pixel/object layers, integrated chat, and map import/export. Built as a group project for the Systems Programming course.

---

### 📚 Course Summary Sharing Platform
**PostgreSQL · Python · Flask**

Web app for sharing and rating academic course summaries. Includes a gamification system with XP points, leaderboard, and a cosmetic shop. Designed and normalized from scratch with full relational modeling.

---

### 🎮 2D Snowboard Side-Scroller *(in progress)*
**JavaScript · HTML5 Canvas**

A procedurally generated snowboarding game inspired by Alto's Adventure. Features dynamic terrain generation, realistic player physics, and flip mechanics. Personal creative project.

---

### 🤖 Autonomous VEX Robotics Program
**C++ · VEX SDK**

Autonomous navigation and obstacle avoidance system built for the VEX Robotics Competition. Integrated ultrasonic and gyroscopic sensors for real-time control decisions.
